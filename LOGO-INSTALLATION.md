## ğŸ“‹ How to Add Logo to Your Shopify Store

Since the logo is referenced through Shopify's theme settings, you'll need to upload it through the Shopify admin interface.

### Method 1: Via Theme Editor (Easiest)

1. **Upload theme to Shopify first**
   - Compress theme folder to .zip
   - Upload to Shopify Admin > Online Store > Themes

2. **Open Theme Editor**
   - Go to **Online Store > Themes**
   - Click **Customize** on your active theme

3. **Upload Logo**
   - In the theme editor, click **Theme settings** (bottom left)
   - Scroll to **Logo** section
   - Click **Select image**
   - Upload `benylo-logo.png` from your computer
   - Or select it from Files if already uploaded

4. **Adjust Logo Size**
   - Set **Logo width** to **180px** (recommended)
   - Set **Mobile logo width** to **120px**
   - Click **Save**

---

### Method 2: Via Settings > Files

1. **Upload logo file**
   - Go to **Settings > Files**
   - Click **Upload files**
   - Select `benylo-logo.png`
   - Copy the file URL

2. **Add to Theme Settings**
   - Go to **Online Store > Themes**
   - Click **Customize**
   - Click **Theme settings** > **Logo**
   - Select the uploaded logo
   - Save
